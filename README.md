# EAgenda

![Gif de funcionamento](https://i.imgur.com/ejPRfP6.gif)

## Introdução
A e-Agenda é uma solução de agenda eletrônica que permite o gerenciamento de compromissos, 
tarefas e contatos, trazendo mais flexibilidade e organização para o seu dia a dia atarefado.

## Funcionalidades

- **Gerenciamento de contatos**: Adicione, edite e remova contatos facilmente.
- **Gerenciamento de compromissos**: Crie compromissos, defina horários e locais. O sistema avisará sobre conflitos entre horários.
- **Gerenciamento de categorias**: Separe suas despesas em categorias para melhor organização.
- **Gerenciamento de despesas**: Registre suas despesas, defina valores e categorias para um controle financeiro eficaz.
- **Gerenciamento de tarefas**: Crie, edite e remova tarefas, definindo prazos e prioridades.
 
Os dados são armazenados localmente em um arquivo JSON, garantindo que você tenha controle total sobre suas informações.

## Como usar

**Clone o repositório:**
```
https://github.com/P-S-T-Partido-Socialista-do-Thiagao/e-agenda.git
```

**Navegue até a pasta raiz do projeto:**
```
cd e-agenda
```

**Restaure as dependências**

```
dotnet restore
```

**Navegue até a pasta do projeto**

```
cd EAgenda.WebApp
```

**Execute o projeto**

```
dotnet run
```
